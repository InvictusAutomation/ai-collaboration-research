## 分配策略

### 1. 基于能力匹配

```python
class SubAgentAllocator:
    def allocate(self, task, available_agents):
        # 计算匹配度
        scores = []
        for agent in available_agents:
            capability_match = self.calculate_capability(task, agent)
            load_factor = agent.current_load / agent.max_load
            score = capability_match * (1 - load_factor)
            scores.append((agent, score))
        
        # 选择得分最高的
        return max(scores, key=lambda x: x[1])[0]
```

### 2. 基于负载均衡

```python
class LoadBalancedAllocator:
    def allocate(self, task, agents):
        # 选择负载最低的
        min_load_agent = min(agents, key=lambda a: a.current_load)
        return min_load_agent
```

### 3. 基于动态调度

```python
class DynamicAllocator:
    def allocate(self, task, agents):
        # 根据任务复杂度动态选择
        if task.complexity < 0.3:
            return self.select_simple_agent(agents)
        elif task.complexity < 0.7:
            return self.select_medium_agent(agents)
        else:
            return self.select_advanced_agent(agents)
```
