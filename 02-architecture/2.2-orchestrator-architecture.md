## 编排模式

### 1. 中心辐射式

```
              ┌─────────────┐
            ╱ │  Orchestrator│ ╲
        ┌──►│             │───┐ ┌──►
        │   └─────────────┘   │ │
        │                     │ │
     Spoke               Spoke Spoke
```

### 2. 层级式

```
          ┌───────┐
          │ Root  │───► Agent A ──► Sub-A1/A2
          └───────┘
               │
               └──► Agent B ──► Sub-B1/B2
```

### 3. 网状式

```
    ┌───────┐
    │   A   │◄──────►
    └─┬─────┘         │
      │         ┌─────┴─────┐
      └────────►│    B    │◄──────►
                └─────────┘
```

## 动态任务分配

```python
class Orchestrator:
    def assign_tasks(self, tasks, agents):
        assignments = {}
        for task in tasks:
            # 选择最合适的 Agent
            best_agent = self.select_best_agent(task, agents)
            # 检查依赖
            if self.check_dependencies(task, assignments):
                assignments[task.id] = best_agent
        return assignments
    
    def select_best_agent(self, task, agents):
        scores = []
        for agent in agents:
            if agent.can_handle(task):
                score = self.calculate_score(task, agent)
                scores.append((agent, score))
        return max(scores, key=lambda x: x[1])[0]
```
