# 4.3 医疗健康领域的 AI 协作

## 概述

医疗健康领域对人机协作有独特需求：辅助诊断、医学研究、患者管理，同时必须严格遵守 HIPAA 等隐私法规。

## 典型应用案例

### 案例 1: 辅助诊断

```
患者主诉: 持续性头痛 2 周

AI 辅助诊断流程:
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  1. 症状提取                                                   │
│     → 头痛持续时间: 2周                                        │
│     → 头痛性质: 钝痛                                           │
│     → 伴随症状: 无                                             │
│     → 加重因素: 站立时加重                                     │
│                                                                 │
│  2. 鉴别诊断                                                   │
│     → 偏头痛 (30%)                                             │
│     → 紧张性头痛 (25%)                                         │
│     → 低颅压头痛 (20%)                                         │
│     → 其他 (25%)                                               │
│                                                                 │
│  3. 建议检查                                                   │
│     → 头颅 MRI                                                 │
│     → 腰椎穿刺 (如 MRI 正常)                                   │
│                                                                 │
│  4. 治疗建议                                                   │
│     → 建议: 神经内科就诊                                       │
│     → 注意: 警惕警告症状 (视物模糊、肢体无力等)                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```

┌─────────────────────────────────────────────────────────────────┐

│                                                                 │
```
