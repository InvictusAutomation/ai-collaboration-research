## 多 Agent 协作示例

### 投资分析 Agent 团队

```python
# 多 Agent 协作
from autogen_agentchat.teams import GroupChat

# 研究员 Agent
researcher = AssistantAgent(
    name="researcher",
    description="行业研究员，负责收集和分析数据"
)

# 策略师 Agent
strategist = AssistantAgent(
    name="strategist",
    description="投资策略师，负责制定投资方案"
)

# 风险管理员 Agent
risk_manager = AssistantAgent(
    name="risk_manager",
    description="风险管理专家，负责评估风险"
)

# 合规 Agent
compliance = AssistantAgent(
    name="compliance",
    description="合规专员，负责合规审查"
)

# 组建团队
team = GroupChat(
    participants=[researcher, strategist, risk_manager, compliance],
    max_turns=8
)

# 执行任务
result = await team.run(
    task="分析新能源车行业投资机会，给出具体建议"
)
```
