# 5.3 成本控制与性能平衡

## 成本构成

| 成本项 | 占比 | 优化策略 |
|--------|------|---------|
| LLM 调用 | 60% | 缓存、批量 |
| Token 费用 | 25% | 压缩、裁剪 |
| 计算资源 | 10% | 弹性伸缩 |
| 网络传输 | 5% | 边缘计算 |

```
┌────────────────────────────────────────────────────────────────┐

```
