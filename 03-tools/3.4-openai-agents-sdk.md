## 核心功能

### 1. Agent Builder

可视化编排工具：

```yaml
# Agent 配置示例
agent:
  name: "Customer Support Agent"
  model: "gpt-4o"
  
  # 工具
  tools:
    - search_knowledge_base
    - create_ticket
    - refund_order
  
  # 知识库
  knowledge:
    - source: "faq"
      vector_store: "faq-store"
    - source: "product_docs"
      vector_store: "docs-store"
```

### 2. 工具集成

```python
from openai import agents

# 定义工具
@agents.tool
def search_knowledge_base(query: str) -> str:
    """搜索知识库"""
    return "相关答案..."

# 创建 Agent
agent = agents.create(
    name="support",
    instructions="你是一个客服助手...",
    tools=[search_knowledge_base]
)
```

### 3. Guardrails（guardrails）

安全护栏：

```python
from openai import agents

# 定义输入验证
input_validation = agents.guardrails.InputGuardrail(
    validators=[
        agents.validators.LegalConstraint(
            message="我们无法处理法律相关请求"
        ),
        agents.validators.PII(
            action="filter"
        )
    ]
)

# 创建 Agent
agent = agents.create(
    name="support",
    guardrails={
        "input": [input_validation]
    }
)
```

### LLM 节点

```yaml
llm_node:
  model: "gpt-4o"
  temperature: 0.7
  system_prompt: |
    你是一个专业的客服代表...
  max_tokens: 1000
```

### Tools 节点

```yaml
tools_node:
  tools:
    - name: search_kb
      description: 搜索知识库
    - name: create_ticket
      description: 创建工单
    - name: send_email
      description: 发送邮件
```

### Condition 节点

```yaml
condition_node:
  conditions:
    - if: "{{intent}}" == "refund"
      goto: "refund_handler"
    - if: "{{intent}}" == "inquiry"
      goto: "inquiry_handler"
    - else:
      goto: "human_handoff"
```

### Loop 节点

```yaml
loop_node:
  max_iterations: 3
  condition: |
    {{attempts < 3}} AND {{not resolved}}
```

## 评估与优化

### Evals

```python
from openai import agents

# 创建评估
eval = agents.evals.create(
    name="support_eval",
    dataset="support_dataset",
    metrics=["accuracy", "helpfulness", "resolution_rate"]
)

# 运行评估
results = eval.run()
```

### Trace

```python
# 追踪执行
with agents.trace() as tr:
    result = await agent.run(task)
    
# 查看追踪
print(tr.spans)
```

## ChatKit 集成

    agent_id="agent_xxx",

# 嵌入网页
