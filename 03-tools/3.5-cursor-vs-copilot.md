# 3.5 Cursor 与 GitHub Copilot 对比

## 概述

Cursor 和 GitHub Copilot 是当前最流行的两款 AI 编程助手，它们各有特色。

> 参考来源：https://cursor.com/, https://github.com/features/copilot

│  │        Cursor           │    │    GitHub Copilot       │      │

## Cursor 核心特性

### 1. 多 Agent 架构

```
┌─────────────────────────────────────────────────────────────────┐
│                    Cursor 多 Agent 架构                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │                    Command K (Ctrl+K)                       │ │
│  │  • 快速编辑      • 生成代码      • 解释代码             │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │                    Chat (Ctrl+L)                           │ │
│  │  • 对话式      • @文件引用      • @文档               │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │                    Agent Mode                            │ │
│  │  • 自主执行    • 多文件修改    • 测试生成              │ │
│  │  • Bug 修复    • 重构         • 端到端开发            │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │
│  │                    Tab (补全)                             │ │
│  │  • 智能补全    • 整行/函数补全  • 多光标              │ │
│  └───────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. @Mentions

```python
# @Files - 引用文件
@file:src/main.py

# @Docs - 引用文档
@doc:README.md

# @Code - 引用代码
@code:src/utils.ts

# @Git - Git 历史
@git:recent-commits
```

### 3. .cursorrules

```yaml
# .cursorrules
rules:
  - description: "项目使用 React + TypeScript"
    rules:
      - "使用函数式组件"
      - "使用 TypeScript 严格模式"
      - "遵循 Airbnb React 规范"
      
  - description: "代码风格"
    rules:
      - "使用 2 空格缩进"
      - "使用单引号"
      - "分号结尾"
      
  - description: "测试"
    rules:
      - "每个组件需要测试"
      - "使用 React Testing Library"
```

## 性能对比

### 代码生成质量

| 场景 | Cursor | Copilot | 说明 |
|------|--------|---------|------|
| 简单函数 | 95% | 90% | 差异小 |
| 复杂逻辑 | 85% | 70% | Cursor 更强 |
| 测试生成 | 80% | 65% | Cursor 更智能 |
| Bug 修复 | 88% | 60% | Cursor 优势明显 |
| 重构 | 82% | 55% | Cursor 领先 |

### 响应速度

| 指标 | Cursor | Copilot |
|------|--------|---------|
| 补全延迟 | <100ms | <100ms |
| Chat 响应 | 2-3s | 2-3s |
| Agent 任务 | 30-60s | N/A |
