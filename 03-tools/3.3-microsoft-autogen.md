# 3.3 Microsoft AutoGen 深度解析

## 概述

AutoGen 是微软推出的开源多智能体框架，专注于构建生产级的多智能体应用。

## 安装

```bash
# 安装 AutoGen
pip install autogen-agentchat

# 安装扩展
pip install autogen-ext[openai]

# 安装 Studio (可选)
pip install autogenstudio
autogenstudio ui --port 8080
```

## 快速开始

### 基础示例

```python
from autogen_agentchat.agents import AssistantAgent
from autogen_ext.models.openai import OpenAIChatCompletionClient

# 创建 Agent
agent = AssistantAgent(
    name="assistant",
    model_client=OpenAIChatCompletionClient(model="gpt-4o"),
)

# 执行任务
result = await agent.run(task="帮我写一个 Python 函数计算斐波那契数列")
print(result)
```

### 多 Agent 对话

```python
from autogen_agentchat.agents import AssistantAgent, UserProxyAgent
from autogen_agentchat.teams import GroupChat

# 创建多个 Agent
assistant = AssistantAgent(
    name="assistant",
    model_client=OpenAIChatCompletionClient(model="gpt-4o")
)

critic = AssistantAgent(
    name="critic",
    model_client=OpenAIChatCompletionClient(model="gpt-4o"),
    system_message="你是一个代码审查员，提供批评性反馈"
)

# 创建群组对话
team = GroupChat(
    participants=[assistant, critic],
    max_turns=2
)

# 运行
result = await team.run(task="写一个排序算法")
```

## 核心组件

### 1. AssistantAgent

```python
from autogen_agentchat.agents import AssistantAgent

agent = AssistantAgent(
    name="assistant",
    model_client=OpenAIChatCompletionClient(model="gpt-4o"),
    system_message="你是一个有帮助的助手",
    tools=[...],  # 可选：附加工具
)
```

### 2. UserProxyAgent

```python
from autogen_agentchat.agents import UserProxyAgent

user_proxy = UserProxyAgent(
    name="user_proxy",
    human_input_mode="NEVER",  # NEVER / TERMINATE / AUTO
)
```

### 3. GroupChat

```python
from autogen_agentchat.teams import GroupChat
from autogen_agentchat.teams import RoundRobinGroupChat
from autogen_agentchat.teams import SelectorGroupChat

# 轮询模式
team = RoundRobinGroupChat(participants=[agent1, agent2])

# 选择器模式（AI 决定下一个发言者）
team = SelectorGroupChat(
    participants=[agent1, agent2, agent3],
    selector_prompt="选择下一个发言者"
)
```

### 4. SelectorGroupChat

```python
from autogen_agentchat.teams import SelectorGroupChat

team = SelectorGroupChat(
    participants=[
        coder,
        tester,
        reviewer
    ],
    selector_prompt="""根据当前对话，选择最合适的发言者。
    
可用发言者：
- coder: 编写代码
- tester: 编写测试
- reviewer: 审查代码
    
请直接输出发言者名称。"""
)
```

## 工具集成

### 代码执行

```python
from autogen_ext.code_executors import DockerCommandLineCodeExecutor

executor = DockerCommandLineCodeExecutor(
    image="python:3.11"
)

agent = AssistantAgent(
    name="assistant",
    model_client=OpenAIChatCompletionClient(model="gpt-4o"),
    code_executor=executor
)

# Agent 可以执行代码
result = await agent.run(task="计算 1-100 的和")
```

### MCP 集成

```python
from autogen_ext.tools.mcp import McpWorkbench

workbench = McpWorkbench(
    stdio_server_command=["python", "mcp_server.py"]
)

agent = AssistantAgent(
    name="assistant",
    tools=workbench.tools
)
```

## 工作流模式

### 1. 两 Agent 协作

```python
# 开发者 + 审查员
developer = AssistantAgent(
    name="developer",
    model_client=OpenAIChatCompletionClient(model="gpt-4o")
)

reviewer = AssistantAgent(
    name="reviewer",
    model_client=OpenAIChatCompletionClient(model="gpt-4o"),
    system_message="审查代码并提供改进建议"
)

# 循环协作
for _ in range(3):
    code = await developer.run(task="写一个功能")
    feedback = await reviewer.run(task=f"审查: {code}")
```

### 2. 层级式

```python
from autogen_agentchat.teams import MagenticOneGroupChat

# MAgentic One 模式：Leader + Workers
team = MagenticOneGroupChat(
    participants=[worker1, worker2, worker3],
    leader=leader_agent
)
```

### 3. 选择式

```python
# AI 动态选择下一步
team = SelectorGroupChat(
    participants=[researcher, coder, writer],
    allow_repeated_speaker=False
)
```
