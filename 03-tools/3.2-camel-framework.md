# 3.2 CAMEL 框架全面解析

## 概述

CAMEL（Communicative Agents for Multi-Step Task Execution）是当前最领先的开源多智能体框架之一，由超过 100 名研究人员组成的社区维护。

> 参考来源：https://github.com/camel-ai/camel

## 安装与快速开始

```bash
# 安装 CAMEL
pip install camel-ai

# 基础使用
from camel.agents import ChatAgent
from camel.models import ModelFactory
from camel.types import ModelPlatformType

model = ModelFactory.create(
    model_platform=ModelPlatformType.OPENAI,
    model_type=ModelType.GPT_4O
)

agent = ChatAgent(model=model)
response = agent.step("Hello, world!")
print(response.msgs[0].content)
```

## 核心模块

### 1. ChatAgent

```python
from camel.agents import ChatAgent
from camel.models import ModelFactory
from camel.types import ModelPlatformType, ModelType
from camel.toolkits import SearchToolkit

# 创建模型
model = ModelFactory.create(
    model_platform=ModelPlatformType.OPENAI,
    model_type=ModelType.GPT_4O,
    model_config_dict={"temperature": 0.0}
)

# 创建 Agent
agent = ChatAgent(
    model=model,
    tools=[SearchToolkit().search_duckduckgo]
)

# 执行任务
response = agent.step("What is CAMEL-AI?")
print(response.msgs[0].content)
```

### 2. RolePlaying (角色扮演)

```python
from camel.societies import RolePlaying

# 创建角色扮演社会
society = RolePlaying(
    assistant_role="Python Developer",
    user_role="Stock Trader",
    task="开发一个股票交易机器人",
    model_platform=ModelPlatformType.OPENAI,
    model_type=ModelType.GPT_4O
)

# 启动对话
chat_history = society.initiate_chat()
print(chat_history)
```

### 3. Workforce (Agent 团队)

```python
from camel.societies import Workforce

# 创建 Agent 团队
workforce = Workforce()

# 添加不同角色的 Agent
workforce.add_agent(
    name="researcher",
    role="Research Analyst",
    description="研究市场趋势"
)

workforce.add_agent(
    name="coder", 
    role="Python Developer",
    description="实现交易策略"
)

workforce.add_agent(
    name="reviewer",
    role="Code Reviewer",
    description="审查代码质量"
)

# 执行任务
result = workforce.process_task("创建一个量化交易系统")
```

## Agent 类型

| Agent 类型 | 功能 | 使用场景 |
|-----------|------|---------|
| ChatAgent | 基础对话 | 简单问答 |
| RolePlaying | 角色扮演 | 多角色协作 |
| CriticAgent | 批判评估 | 质量把控 |
| WriterAgent | 写作 | 内容生成 |
| AssistantAgent | 助手 | 通用任务 |

## 工具集成

### 内置工具

```python
from camel.toolkits import SearchToolkit
from camel.toolkits import BrowserToolkit
from camel.toolkits import CodeExecutionToolkit

# 搜索工具
search = SearchToolkit()
results = search.search_duckduckgo("Python 教程")

# 浏览器工具
browser = BrowserToolkit()
page = await browser.navigate("https://example.com")

# 代码执行
executor = CodeExecutionToolkit()
result = executor.execute_python("print('Hello')")
```

### 自定义工具

```python
from camel.tools import BaseTool
from camel.types import ToolCall

class MyCustomTool(BaseTool):
    name = "my_custom_tool"
    description = "自定义工具描述"
    
    def forward(self, input: str) -> str:
        return f"处理: {input}"
```

## 记忆系统

```python
from camel.memories import AgentMemory
from camel.storages import RedisStorage

# 创建记忆
memory = AgentMemory(
    storage=RedisStorage(
        host="localhost",
        port=6379
    )
)

# 添加记忆
memory.add("用户偏好: 喜欢简洁的代码风格")

# 检索记忆
relevant = memory.retrieve("代码风格")
```

## RAG 集成

```python
from camel.rag import RAG

# 创建 RAG
rag = RAG(
    vector_store="chromadb",
    embedding_model="sentence-transformers"
)

# 添加文档
rag.add_documents([
    "文档1内容...",
    "文档2内容..."
])

# 检索
results = rag.retrieve("查询内容")
```

## 应用场景

### 1. 数据生成

```python
# 生成训练数据
from camel.datagen import SelfInstruct

generator = SelfInstruct(
    domain="programming",
    num_samples=1000
)

dataset = generator.generate()
```

### 2. 任务自动化

```python
# 自动化工作流
from camel.societies import Workflow

workflow = Workflow(
    steps=[
        {"agent": "researcher", "task": "收集信息"},
        {"agent": "analyzer", "task": "分析数据"},
        {"agent": "writer", "task": "生成报告"}
    ]
)

result = workflow.execute()
```

### 3. 世界模拟

```python
# 多 Agent 模拟
from camel.simulations import WorldSimulation

simulation = WorldSimulation(
    num_agents=100,
    environment="city"
)

simulation.run(duration="1h")
```
