## 腾讯 CodeBuddy

> 参考来源：https://codebuddy.tencent.com/

### 核心特性

| 特性 | 说明 |
|------|------|
| 双层 SPEC | HumanSpec + AISpec 架构 |
| Context 管理 | Memory、Rules、Skills、MCP、Integrations |
| Agent 工作流 | Project → Chat → Request → Agent → Tools |
| 腾讯生态 | 深度集成微信、企业微信、TAPD |

### 与国际产品对比

| 维度 | CodeBuddy | Claude Code | Cursor |
|------|-----------|-------------|--------|
| SPEC 驱动 | ✅ 独有 | ❌ | ❌ |
| 本土化 | ✅ 腾讯系 | ❌ | ❌ |
| 多 Agent | ✅ | ✅ | ✅ |
| 价格 | 待定 | $20/月 | $20/月 |

## 新兴开源项目

### Lagent

```python
# Lagent 是一个轻量级 Agent 框架
from lagent import Agent

agent = Agent(
    model='internlm',
    plugins=['python', 'search']
)

result = agent('计算斐波那契数列')
```
